# Master_Thesis_UKBiobank

Work-in-progress codebase for my MSc thesis on PTSD classification using resting-state fMRI from the UK Biobank. This repository currently contains three main components: feature extraction scripts, two shallow ML baselines, and simplified graph-based models for later sample-level modelling. 

## Overview

- **Feature extraction** full versus partial correlations from atlas- and ICA-derived rs-fMRI representations
- **Shallow baselines** for classification, including SVM and Elastic Net
- **Graph-based models** used to prototype spatio-temporal sample-level modelling (WIP)

At the moment, this is a research code repository rather than a packaged software release. The code reflects an active thesis workflow and is being cleaned up and reorganized over time.
